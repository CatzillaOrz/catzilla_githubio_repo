###### 创建自己的GitHub图床

关键步骤： 生成一个Token用于操作GitHub repository

###### vs code插件

picgo

将github仓库 + 密钥，天蝎到picgo插件配置项中，然后就可以在vscode里，随时上传图片（只需复制图片到剪切板即可）

Mac OS快捷键

```bash
    Cmd + Opt + U
```
